## 개발 기능

- 인사 관리(사용자, 부서)
    - 인적사항관리
    - 인사발령
    - 인사현황
    - 조직도관리
- 근태 관리
    - 근태이력관리
    - 근무일정/시간
    - 초과 근무
- 급여 관리
    - 급여이력조회
    - 성과급 관리
- 업무 관리
    - 업무 조회(진행사항 등)
    - 업무 배정
- 성과 관리
    - 업적 평가
    - 역량 평가
    
## 의존 라이브러리

- data-jpa
- thymeleaf
- spring-web
- security
- lombok
- devtools
- h2 database

## 개발 환경

- Gradle Project
- Spring Boot 2.4.1
- Java jdk 11
